# Task 5 - Exploratory Data Analysis (EDA)

## Objective

Perform exploratory data analysis on the Titanic dataset to identify patterns, trends, and relationships using statistical summaries and visualizations.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

Titanic Dataset (train.csv)

## Analysis Performed

### Data Exploration

* Dataset Information
* Statistical Summary
* Missing Value Analysis

### Univariate Analysis

* Survival Distribution
* Gender Distribution
* Age Distribution

### Bivariate Analysis

* Survival by Gender
* Survival by Passenger Class
* Fare vs Survival

### Multivariate Analysis

* Correlation Heatmap
* Pairplot

## Key Findings

* Females had higher survival rates than males.
* First-class passengers survived more frequently.
* Higher fares were associated with higher survival rates.
* Missing values were primarily present in Cabin and Age columns.
